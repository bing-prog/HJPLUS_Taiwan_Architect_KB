# 文化資產保存法

本知識領域依據《文化資產保存法》及《古蹟修復及再利用辦法》等相關法規，提供台灣建築師在處理歷史建築、古蹟及聚落建築群等文化資產專案時的專業指引。

文化資產專案具有特殊性，其作業程序（如：調查研究、修復設計、監造與工作報告）與一般建築專案有顯著差異。此外，《文資法》對於《建築法》、《都市計畫法》、《消防法》等相關規定設有排除條款，透過製作「再利用因應計畫」經主管機關核定後，即可不受原法規之限制，從而保障文化資產之真實性與風貌。

## 涵蓋範圍

- **文化資產項目**：文化資產的定義與分類，包含古蹟、歷史建築、紀念建築及聚落建築群等。
- **調查研究**：進入文化資產實質修復設計前，必須先進行歷史考證、測繪及現況損壞調查。
- **修復設計**：基於「調查研究」成果所進行之實質修復工程設計，遵循真實性原則。
- **監造與工作報告**：文化資產修復工程施工期間的監督與紀錄作業，完工後須繳交工作報告書。
- **再利用因應計畫**：針對文資再利用所面臨之建管、土地使用及消防安全衝突，研擬替代方案之程序與規範。
- **權益補償**：私有文化資產因受法令限制或鼓勵保存所提供之補償機制，包含地價稅及房屋稅等稅賦減免，以及古蹟與歷史建築之容積移轉作業。
