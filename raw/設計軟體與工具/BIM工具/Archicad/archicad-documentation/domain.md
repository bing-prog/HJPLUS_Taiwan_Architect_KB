# Archicad 圖面與文件產出

## 為什麼重要？

Archicad 的出圖品質不只取決於模型，也取決於視圖、版面、標註、圖框與發布設定。若直接從原始視點出圖，或沒有固定的 View Map 與 Layout Book 管理方式，圖面很容易出現比例、圖層、筆色、圖號與修訂資訊不一致的問題。

## 文件產出流程

```text
模型視點 -> 儲存視圖 -> 放入圖紙 -> 發布輸出
```

| 階段 | 對應位置 | 重點 |
|---|---|---|
| 模型視點 | Project Map | 平面、立面、剖面、詳圖、3D、表格 |
| 儲存視圖 | View Map | 比例、圖層組合、筆色、顯示方式 |
| 圖紙配置 | Layout Book | 圖框、圖名、圖號、比例、版面 |
| 發布輸出 | Publisher Sets | PDF、DWG、BIMx、資料夾與檔名 |

## 出圖原則

- 正式圖面應從 View Map 放到 Layout，不建議直接使用未儲存的視點。
- 同一張平面若有不同比例或不同用途，應建立不同視圖。
- 圖名、圖號、比例與圖框資訊應盡量由欄位或圖紙資料帶入。
- 尺寸標註與標籤應盡量連結模型資訊，減少人工重複輸入。
- 出圖前應更新所有 Drawing，避免版面中的圖面不是最新狀態。

## 常見輸出類型

| 類型 | 用途 |
|---|---|
| PDF | 正式送審、提案、施工圖或內部審查 |
| DWG | 與顧問、承包商或其他 CAD 流程交換 |
| BIMx | 模型瀏覽、業主溝通、現場檢視 |
| 圖片 | 簡報、快速檢討或溝通 |
| IFC | BIM 協調與模型交換，不是正式圖面格式 |

## 發布前檢查清單

- 所有圖面是否已更新？
- 圖紙順序與圖號是否正確？
- 圖名、比例、日期、版次是否一致？
- 標註是否被裁切或遮蔽？
- PDF 線寬是否清楚？
- DWG 是否能在其他檢視器中正常開啟？
- 發布檔名是否含有日期、案名、圖種或版次？
